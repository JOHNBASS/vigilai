VigilAI POC

用一句話，讓 AI 守望你的鏡頭。
打開 webcam，用自然語言描述要偵測什麼，命中時即時推播到 Telegram。

核心功能

  • 自然語言定義偵測　寫一句 prompt 就能偵測，免訓練、免標註資料
  • 即時 Webcam 分析　瀏覽器開鏡頭，每隔數秒由 VLM（qwen2.5-VL）看畫面
  • Telegram 即時通知　命中時把「觸發截圖 + 說明」推到你手機
  • 數值 / 布林雙模式　OCR 讀數字做門檻比較，或是/否事件偵測
  • 冷卻防洪水　每條規則可設 cooldown，避免連續洗版告警

運用場景

🌡️ 高溫警報（溫度 ≥ 門檻） 🔥 火災 / 濃煙偵測 😴 打瞌睡 / 疲勞偵測 ✋ 手勢辨識 🚸 入侵 / 有人偵測 🏭 產線瑕疵檢查 👶 嬰幼兒 / 寵物看護 📟 儀表板讀值監看
